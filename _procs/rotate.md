## Documentation

```
;;; (rotate img angle [description]) -> image?
;;;   img : image?
;;;   angle : real?
;;;   description : string?
;;; Create a new image by rotating `img` by `angle` degrees
;;; counter-clockwise.
;;;
;;; Note that this procedure may trim some of the transparent
;;; parts of an image after rotating.
```
